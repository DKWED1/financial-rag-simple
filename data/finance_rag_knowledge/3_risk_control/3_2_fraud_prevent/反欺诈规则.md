# 风险防控风控规则
## 一、风险防控总体原则
1.  平台建立多维度风险防控风控体系，通过设备、行为、关联关系、外部数据等维度识别违规风险。
2.  风险防控规则分为设备风险、行为风险、关联风险、外部数据风险四大模块，任一模块命中高风险规则，直接拒绝申请。
3.  风险防控命中记录将永久保留在风控档案中，影响用户后续授信与准入。
4.  平台与第三方风险防控机构合作，共享违规黑名单，提升风险识别能力。

## 二、设备风险识别规则
1.  设备异常：同一设备/手机/IP地址关联≥5个不同身份证号申请，标记为高风险。
2.  模拟器/虚拟机申请：使用模拟器、虚拟机、云手机申请，直接拒绝。
3.  网络代理IP：使用网络代理、中转IP、境外IP申请，直接拒绝。
4.  虚拟定位：使用虚拟定位、GPS修改软件申请，直接拒绝。
5.  设备权限篡改：设备权限破解、系统越狱状态申请，标记为高风险，从严审批。
6.  同一IP高频申请：同一IP地址24小时内出现≥3笔不同用户申请，标记为批量组团申请风险。

## 三、申请行为风险识别规则
1.  申请频率过高：24小时内同一用户申请≥3次，标记为高风险。
2.  申请时间异常：凌晨（0:00-6:00）高频申请，标记为高风险。
3.  填写信息不一致：姓名、手机号、地址前后填写不一致，标记为虚假填报申请风险。
4.  填写信息与外部数据不符：工作单位、地址、收入与第三方数据核验不符，标记为虚假填报申请。
5.  材料伪造：上传的身份证、流水、工作证明经核验为伪造，直接拒绝并拉入黑名单。
6.  人脸核验不通过：人脸核验失败≥3次，标记为盗用他人身份风险，直接拒绝。

## 四、关联关系风险识别规则
1.  亲属关联违规：申请人直系亲属存在恶意欠款、拒不履约记录，标记为高风险。
2.  紧急联系人违规：申请人紧急联系人存在平台黑名单记录，标记为关联风险。
3.  组团申请关联：同一地址、同一IP、同一设备关联多个申请用户，标记为组团违规风险。
4.  联系人异常：申请人联系人中存在大量平台黑名单用户，标记为关联风险。
5.  设备关联黑名单：申请人使用的设备曾被平台其他违规用户使用，标记为高风险。

## 五、外部数据风险识别规则
1.  第三方风险防控黑名单命中：直接拒绝申请，永久禁止准入。
2.  多头授信风险：近3个月在≥10家金融机构申请授信，标记为以贷周转风险。
3.  线上授信逾期黑名单命中：在其他平台存在逾期拒不履约记录，直接拒绝。
4.  司法失信记录命中：被列入失信被执行人名单，直接拒绝。
5.  涉公共安全涉案记录命中：涉及网络涉诈、非法募资、层级营销等案件，直接拒绝。

## 六、风险等级分级处理规则
### 高风险违规（直接拒绝+永久黑名单）
- 命中平台/第三方违规黑名单
- 材料伪造、盗用他人身份申请
- 组团申请、批量恶意授信
- 司法涉案、金融违规记录

### 中风险违规（从严审批+限制额度）
- 设备异常、高频申请
- 多头授信严重
- 信息填写不一致、核验不符
- 人脸核验多次失败

### 低风险违规（人工复核+重点监控）
- 异地申请、非工作时间申请
- 同一IP少量申请
- 手机号入网时间较短

## 七、风险处置与留痕规则
1.  所有风险防控命中记录全程留痕，保存≥5年，用于合规审计与风控分析。
2.  命中高风险违规规则的用户，直接拉入平台永久黑名单，不得再次申请。
3.  中低风险违规用户，申请将被人工复核，核实为违规的，列入灰名单，限制后续申请。
4.  涉嫌违法犯罪的违规行为，平台将向公安部门报备，配合调查取证。
5.  风险防控风控规则定期更新，根据新型违规手段优化识别模型。
