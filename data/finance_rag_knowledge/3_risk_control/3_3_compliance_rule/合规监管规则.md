# 合规监管规则
## 一、合规总体原则
1.  平台所有信贷业务严格遵守国家金融监管政策，不得违反《商业银行互联网贷款管理暂行办法》《关于规范整顿业务的通知》等监管规定。
2.  平台利率、费率严格控制在年化24%以内，不得存在前置收费、不合规营销、过激客诉处置等违规行为。
3.  平台所有用户信息、业务数据严格按照监管要求进行存储、使用与保护，保障用户信息安全。
4.  平台定期接受监管部门检查，及时整改不合规业务，确保业务合法合规运营。

## 二、利率与费用合规规则
1.  综合年化利率上限：平台所有信贷产品综合年化利率不得超过24%，包含利息、服务费、罚息、违约金等所有费用。
2.  禁止隐性收费：禁止收取合同外的任何费用，如前置服务费、保证金、押金、额外服务费等。
3.  利率展示合规：借款前必须明确展示年化利率、日利率、总利息、每期还款额，不得使用“日息低至XX元”等模糊表述。
4.  罚息与违约金合规：罚息利率不得超过合同约定利率的1.5倍，且不得计入复利后突破年化24%上限。
5.  禁止诱导消费：不得通过夸大宣传、虚假优惠诱导用户过度借贷。

## 三、用户信息保护合规规则
1.  信息采集合规：仅采集与信贷审批、风控审核相关的必要信息，不得过度采集用户隐私信息。
2.  信息使用合规：用户信息仅用于信贷审批、风控审核、后续资产管理，不得向第三方泄露或出售。
3.  信息存储合规：用户敏感信息（身份证、银行卡、人脸信息）采用加密存储，定期备份，防止数据泄露。
4.  用户授权合规：采集、使用用户信息必须获得用户明确授权，用户可随时撤回授权。
5.  信息删除合规：用户申请注销账户后，按监管要求删除或匿名化处理用户信息。

## 四、客诉沟通合规规则
1.  沟通时间合规：沟通仅可在8:00-22:00进行，禁止非工作时段打扰用户。
2.  沟通方式合规：禁止言语攻击、辱骂、威胁、恐吓用户及第三方联系人。
3.  第三方联系合规：不得打扰用户无关第三方，除非用户失联且经内部审批。
4.  沟通信息合规：不得伪造官方文书、冒用公职机关名义进行沟通。
5.  沟通留痕合规：所有沟通行为必须全程录音/截图留痕，保存≥5年，以备监管检查。

## 五、资金用途合规规则
1.  禁止流入楼市：资金不得用于购房、首付、偿还房贷等房地产相关用途。
2.  禁止流入股市：资金不得用于炒股、买基金、期货、数字资产等投资用途。
3.  禁止流入博彩：资金不得用于博彩、彩票、非法竞猜等用途。
4.  禁止流入非标借贷：资金不得用于偿还非标借贷、其他平台负债。
5.  用途监控合规：平台有权监控资金流向，发现违规用途的，可提前收回授信资金。

## 六、禁止向特定人群授信规则
1.  禁止向全日制在校学生提供授信，包含本专科、研究生、博士生群体。
2.  禁止向无稳定收入来源的人群提供授信，如无业、长期待业人员。
3.  禁止向未成年人提供授信，年满18周岁但无稳定收入的也不予授信。
4.  禁止向被列入失信被执行人、限制高消费名单的人群提供授信。
5.  禁止向从事非法金融活动、高风险行业的人群提供授信。

## 七、业务数据上报合规规则
1.  征信上报合规：逾期用户按监管要求如实上报征信系统，不得延迟、隐瞒上报。
2.  监管数据上报：按监管部门要求定期上报业务数据，包含放款金额、逾期率、坏账率等。
3.  数据备份合规：业务数据、风控记录、沟通记录定期备份，保存期限不低于监管要求。
4.  审计合规：平台内部定期开展合规审计，外部审计机构按监管要求进行专项审计。

## 八、违规处置与整改规则
1.  发现违规业务立即暂停，按监管要求整改，整改完成后方可恢复业务。
2.  违规利率、费用立即调整至合规范围内，对已收取的违规费用按规定退还用户。
3.  违规沟通行为立即停止，对违规工作人员进行内部处罚，情节严重的移交相关机关。
4.  信息泄露事件立即启动应急处置，通知受影响用户，并向监管部门报告。
5.  平台建立合规整改台账，跟踪整改情况，定期向监管部门汇报。
