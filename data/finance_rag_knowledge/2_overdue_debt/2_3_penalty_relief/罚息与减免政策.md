# 逾期费用与减免政策
## 一、费用计算标准
1.  罚息利率：按合同约定利率上浮50%计算，日利率=合同约定日利率×1.5
2.  计算基数：当期未偿还的本金+利息+服务费全额
3.  计算方式：费用=逾期金额×罚息日利率×逾期天数
4.  计息规则：按日计息，按月**累计计息**，计入下一期应还款项
5.  上限控制：各类费用合计，不得超过**法定上限**（按监管要求）

## 二、违约金计算标准
1.  违约金比例：按当期未还款金额的5%收取，最低50元，最高不超过当期应还款项
2.  收取条件：客户逾期超过15天，系统自动计收违约金
3.  上限控制：违约金与罚息合计，不得超过**法定上限**
4.  一次性收取：每笔逾期仅收取一次违约金，不重复计收

## 三、费用的减免条件
### （一）可申请减免的场景
1.  客户因重大疾病、意外事故、自然灾害等不可抗力导致逾期，提供证明材料可申请减免
2.  客户因**特殊环境**、失业等特殊情况导致暂时还款困难，提供失业证明、收入证明可申请减免
3.  客户为首次逾期、逾期天数≤15天，且已全额结清欠款，可申请减免部分费用
4.  客户为优质老用户（历史还款记录良好，无逾期记录），可申请减免违约金
5.  客户已全额结清逾期欠款，且后续无再次逾期，可申请部分费用减免

### （二）不可申请减免的场景
1.  恶意逾期、**恶意拖欠**、故意不还款的客户，一律不予减免
2.  多次逾期、历史逾期记录不良的客户，不予减免
3.  客户逾期超过60天，且未结清欠款的，不予减免
4.  客户申请减免时无法提供有效证明材料的，不予减免
5.  客户存在多笔逾期、**多平台借贷**的，不予减免

## 四、减免申请流程与审批
1.  客户需通过APP、客服渠道提交减免申请，并上传相关证明材料
2.  贷后审核人员在3个工作日内完成材料审核，核实逾期原因与客户资质
3.  审批通过后，系统自动减免对应费用，并更新客户账单
4.  审批未通过的，需告知客户拒绝理由，并同步后续还款方案
5.  减免申请仅可申请一次，同一笔逾期不得重复申请减免

## 五、特殊减免政策
1.  **特殊时期**，可根据监管政策临时放宽减免条件，推出专项减免活动
2.  针对逾期结清的客户，后续复借可享受部分费用减免权益，作为客户关怀
3.  客户主动协商还款、制定还款计划并按时履约的，可申请部分费用减免
4.  **风险资产**处置阶段，客户一次性结清全部欠款的，可申请减免全部费用

## 六、减免后的客户权益与限制
1.  减免仅针对本次逾期的费用，不影响客户的逾期记录上报征信
2.  减免后客户需按约定还款计划履约，再次逾期将取消所有减免资格
3.  减免后的客户后续授信审批将受到一定限制，需观察后续还款表现
4.  恶意申请减免、提供虚假材料的客户，将被列入**风险关注名单**，永久禁止减免申请
