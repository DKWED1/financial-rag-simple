# 个人消费贷授信额度审批规则

## 一、授信额度总体原则
1. 授信额度基于用户身份资质、收入水平、征信情况、负债情况、工作稳定性综合评估。
2. 额度区间为固定等级制，系统自动评级授信，人工仅做异常复核。
3. 同一用户在平台仅允许拥有一个有效授信额度，不叠加、不重复授信。
4. 授信额度有效期为12个月，到期后需重新进行风险评估与额度重审。

## 二、基础授信额度区间（默认等级）
1. A级优质客户：授信额度 50000元 - 200000元
2. B级良好客户：授信额度 30000元 - 50000元
3. C级普通客户：授信额度 10000元 - 30000元
4. D级受限客户：授信额度 3000元 - 10000元
5. E级禁止授信：直接拒绝额度，不予通过

## 三、收入与额度对应规则
### 1. 在职人员（工资打卡）
1. 月收入 3000 - 5000元：授信额度 3000 - 10000元
2. 月收入 5000 - 8000元：授信额度 10000 - 30000元
3. 月收入 8000 - 15000元：授信额度 30000 - 80000元
4. 月收入 15000元以上：授信额度 50000 - 200000元
5. 月收入低于当地最低工资标准：直接禁止授信

### 2. 自雇/个体工商户
1. 月均流水 5000 - 10000元：额度 3000 - 10000元
2. 月均流水 10000 - 30000元：额度 10000 - 30000元
3. 月均流水 30000 - 80000元：额度 30000 - 80000元
4. 月均流水 80000元以上：额度 50000 - 200000元

## 四、征信状况对额度影响规则
1. 征信无逾期、无异常、查询正常：额度上浮10%-30%
2. 近6个月有1次轻微逾期（≤3天）：额度下浮30%
3. 近12个月逾期次数≥2次或逾期>30天：额度下浮50%-80%
4. 近3个月**审批查询**次数 4-6次：额度下浮20%
5. 近3个月**审批查询**次数 7-10次：额度下浮50%
6. 近3个月**审批查询**次数>10次：禁止授信
7. 征信显示呆账、冻结、核销、代偿：禁止授信

## 五、负债与额度扣减规则
1. 负债率 ≤30%：额度不受影响，可按标准执行
2. 负债率 30%-50%：额度下浮20%
3. 负债率 50%-70%：额度下浮50%
4. 负债率 >70%：禁止授信
5. 未结清贷款笔数 3-5笔：额度下浮30%
6. 未结清贷款笔数 >5笔：禁止授信
7. 信用卡使用率 >70%：额度下浮40%

## 六、工作稳定性影响规则
1. 现单位工作满 2年以上：额度上浮20%
2. 现单位工作满 1-2年：额度上浮10%
3. 现单位工作满 6-12个月：标准额度，不浮动
4. 现单位工作 <6个月：额度下浮30%
5. 无固定职业、灵活就业无流水：禁止授信
6. 高危行业、禁入行业从业人员：禁止授信

## 七、资产增信额度提升规则
1. 本人名下商品房/安置房：额度上浮20%-40%
2. 本人名下非运营车辆：额度上浮10%-20%
3. 公积金连续缴存 12个月以上：额度上浮30%
4. 社保连续缴存 12个月以上：额度上浮20%
5. 持有银行定期/理财/基金资产：额度上浮10%-30%
6. 平台历史优质还款用户：额度上浮20%-50%

## 八、年龄与额度区间规则
1. 年龄 22-25周岁：最高额度不超过50000元
2. 年龄 26-35周岁：额度无上限，按综合资质评定
3. 年龄 36-45周岁：额度无上限，按综合资质评定
4. 年龄 46-50周岁：最高额度不超过100000元
5. 年龄 51-55周岁：最高额度不超过50000元
6. 年龄超出22-55周岁：禁止授信

## 九、额度冻结、调减、取消规则
1. 逾期发生时：立即冻结额度，禁止再次支用
2. 征信出现新不良记录：额度直接调减50%-100%
3. 多头借贷突然增加：额度冻结并重新评估
4. 失联、虚假信息、欺诈风险：额度直接清零
5. 政策调整、区域下架：额度自动失效
6. 账户异常、设备风险、IP异常：额度临时冻结

## 十、额度支用规则
1. 单笔支用最低 1000元，最高不超过剩余授信额度
2. 支持循环支用，还款后额度自动恢复（无逾期情况下）
3. 支用期限支持 3/6/9/12 期，按审批等级开放可选期数
4. 额度需在有效期内支用，过期自动失效
5. 禁止用于购房、**证券投资**、**金融投资**、**偿还债务**、**非法赌博**等违规用途
6. 违规支用将触发额度提前结清，并计入**风险名单**
