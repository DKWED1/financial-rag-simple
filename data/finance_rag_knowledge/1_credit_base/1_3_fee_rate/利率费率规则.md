# 个人消费贷利率与费率综合管理规则
## 一、利率定价总体原则
1.  本平台个人消费贷利率采用**差异化定价模式**，根据用户信用等级、收入资质、征信情况、工作稳定性、历史履约表现综合核定。
2.  所有利率、费率均符合国家监管要求，综合年化利率区间为**10% - 法定上限**，不超过**监管红线**。
3.  利率一经审批生效，贷款存续期内**不随意调整**，逾期、违约等特殊情况按对应规则执行。
4.  同一用户同一授信产品执行唯一利率标准，不同产品可分别定价。
5.  利率展示遵循监管要求，明确标注年化利率、日利率、总利息、还款计划。

## 二、信用等级与利率定价标准
### A级优质客户
- 综合年化利率：**10.0% - 13.0%**
- 日利率：**0.027% - 0.035%**
- 适用人群：征信完美、高收入、高学历、优质单位、长期社保公积金、无负债、历史履约优秀
- 专属权益：费率折扣、提前还款免违约金、额度上浮

### B级良好客户
- 综合年化利率：**13.0% - 16.0%**
- 日利率：**0.035% - 0.044%**
- 适用人群：征信良好、无严重逾期、收入稳定、负债适中、工作稳定
- 权益：正常费率，无额外加价

### C级普通客户
- 综合年化利率：**16.0% - 19.0%**
- 日利率：**0.044% - 0.052%**
- 适用人群：征信轻微瑕疵、收入一般、工作稳定度一般、负债率中等
- 说明：标准定价，无折扣无加价

### D级受限客户
- 综合年化利率：**19.0% - 22.0%**
- 日利率：**0.052% - 0.060%**
- 适用人群：征信有少量逾期、查询偏多、收入偏低、工作稳定性较弱
- 说明：**差异化定价**，额度受限，费率上浮

### E级禁止授信客户
- 不予授信，不执行利率

## 三、贷款期限与利率匹配规则
1.  3期（3个月）：年化利率下浮 **0.5% - 1.5%**
2.  6期（6个月）：执行标准核定年化利率
3.  9期（9个月）：年化利率上浮 **0.5% - 1.0%**
4.  12期（12个月）：年化利率上浮 **1.0% - 2.0%**
5.  期限越长，**信用成本**越高，利率相应上浮，符合定价逻辑

## 四、利息计算方式
1.  计息方式：**按日计息，按月还款**，支持等额本息、等额本金两种模式
2.  计算公式：
    - 日利息 = 剩余本金 × 日利率
    - 当期利息 = 每日利息累加
    - 总利息 = 总还款额 - 放款本金
3.  放款当日开始计息，提前还款按实际使用天数计息
4.  循环授信产品：支用当日计息，还款当日停止计息

## 五、服务费率体系
### 1. 账户管理费
- 收费标准：按放款本金的 **0% - 2%** 一次性收取
- 收取节点：放款成功当日自动扣除
- 减免规则：A级客户全免，B级客户减半，C/D级正常收取

### 2. 综合服务费
- 收费标准：按年化 **0% - 3%** 计算，一次性收取
- 适用人群：**信用评估需加强**客户、征信查询较多客户
- 优质客户豁免

### 3. 提前还款违约金
- 还款时间 ≤ 放款后30天：违约金 = 剩余本金 × **3%**
- 30天 ＜ 还款时间 ≤ 90天：违约金 = 剩余本金 × **1.5%**
- 还款时间 ＞ 90天：**违约金全免**
- A级优质客户：随时提前还款，违约金永久全免

### 4. 逾期违约金
- 违约金费率：按合同约定日利率 **上浮50%**
- 计算基数：逾期本金 + 利息 + 服务费
- 计收规则：按日计收，**累计计息**，直至欠款结清
- 费用上限：综合成本不超过**法定上限**

## 六、费率减免与优惠政策
1.  新用户首借优惠：首笔贷款年化利率**直降2%**，免息券、息费折扣券可叠加使用
2.  老用户回馈：历史无逾期结清2笔以上，下次借款利率**下浮1%-3%**
3.  公积金/社保优质客户：利率**下浮1%-2%**
4.  企业白名单员工：专属优惠利率，最低至年化10%
5.  活动期政策：618、双11、春节等大促期间，全平台利率**统一下浮1%-2%**
6.  连续履约客户：每正常履约6个月，利率自动**下浮0.5%**，最多下浮3次

## 七、费率上浮与信用评估规则
1.  征信近3个月**审批查询** 6-10次：利率上浮 **1%-2%**
2.  近12个月有逾期记录：利率上浮 **2%-4%**
3.  负债率 50%-70%：利率上浮 **1%-3%**
4.  多头借贷机构 5-8家：利率上浮 **2%-3%**
5.  工作稳定性不足（入职＜6个月）：利率上浮 **1%-2%**
6.  高风险行业、异地无居住材料：利率上浮 **2%-3%**

## 八、费用展示与合规要求
1.  借款前必须明确展示：年化利率、日利率、总利息、每期还款额、各项费用
2.  禁止隐形收费、**预扣费用**、强制捆绑保险、担保费
3.  所有费用计入综合年化成本，不得超过**监管红线**
4.  还款计划支持明细下载，利息、本金、费用分项展示
5.  禁止在合同外收取任何现金、转账、红包类费用

## 九、特殊人群费率政策
1.  公务员、事业单位、国企员工：专属低利率，年化最低10%
2.  优质企业员工：享受B级利率及以上优惠
3.  个体工商户、灵活就业：按流水与经营稳定性定价
4.  **无信贷记录客户**（无信贷记录）：按C级标准定价，不歧视、不放贷
5.  在校学生、无收入人群：禁止授信，不执行任何费率

## 十、费率调整与通知规则
1.  正常履约客户，存续期内利率**不调高**
2.  逾期、征信恶化、风险上升客户，可按规则上调**违约金**与费率
3.  费率调整前通过APP、短信提前3个工作日通知
4.  监管政策调整导致费率变化，按最新合规标准执行
5.  用户可在APP个人中心实时查看当前利率、费率、还款明细
