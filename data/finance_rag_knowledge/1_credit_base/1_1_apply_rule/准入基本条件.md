# 准入基本条件（个人消费贷）
## 1. 主体资格条件
1.  年龄要求：申请人年龄须在 **22周岁（含）至55周岁（含）** 之间，不含港澳台及外籍人士。
2.  国籍与居住：须为中国大陆居民，持有有效期内二代居民身份证；在申请地有稳定居住地址，可提供居住证明（如近3个月水电煤账单、租赁合同等）。
3.  职业与收入：
    -  在职人员：须在当前单位连续工作满 **6个月以上**，月收入不低于当地最低工资标准的 **2倍**（以打卡工资流水或个税证明为准）。
    -  自雇人士：须持有营业执照满 **1年以上**，近6个月对公账户月均流水≥5000元。
4.  征信状态：
    -  当前无逾期，近6个月逾期次数≤1次，逾期天数≤30天；
    -  近12个月无连三累六（连续3期或累计6期逾期）；
    -  近3个月征信查询次数（硬查询，如贷款审批、信用卡审批）≤6次；
    -  无呆账、止付、冻结等异常账户状态。

## 2. 授信准入补充规则
1.  负债要求：申请人月均还款额（含本笔贷款）≤月收入的 **50%**，整体资产负债率≤70%。
2.  多头借贷限制：近3个月新增贷款机构数≤3家，未结清贷款笔数≤5笔。
3.  地域限制：仅支持中国大陆境内除新疆、西藏、青海部分地区外的申请人（受当地合规要求影响）。

## 3. 其他限制条款
1.  不得为在校学生、无稳定收入来源人员申请；
2.  不得为从事高风险行业（如金融中介、**娱乐业**、高污染化工等）人员申请；
3.  申请人需同意平台进行**个人信用报告**查询、第三方数据核验及电话回访；
4.  本条件最终解释权归平台所有，将根据**合规要求**及**业务策略**动态调整。
