# 申请材料要求（全量正式版）
## 一、整体总则规范
1. 所有贷款申请材料必须为申请人本人真实有效原件影像，不允许翻拍、截图、PS修改、借用他人资料代办。
2. 全部材料上传格式仅限 JPG、PNG 标准图片格式，单张文件大小不得超过5MB，无反光、无遮挡、无水印、无模糊、无裁切不全。
3. 所有填报信息、上传材料、人脸信息必须三方一致，信息不一致直接初审驳回，记入**风险关注名单**。
4. 材料一经提交，平台自动留存归档，用于征信核验、**合规审核**、贷后溯源、审计，不予退回。
5. 材料不实、虚假填报、提供虚假证明，直接命中**禁入名单**，永久禁止准入。

## 二、核心身份必备材料（缺一不可）
1. 本人有效期内二代居民身份证原件正反面完整实拍图，边框齐全，芯片区域无遮挡。
2. 本人实名手机号码，必须本人实名入网满6个月以上，正常在网，非虚拟号、非副卡、非物联网卡。
3. 本人名下一类借记储蓄卡，支持主流银行，不支持二类卡、三类电子账户、社保卡、公积金卡、信用卡收款。
4. 实时人脸活体检测，本人现场出镜，无美颜、无口罩、无帽子、无眼镜遮挡，真人动态核验。
5. 实名短信核验，本人手机实时接收验证码，完成实名绑定核验。

## 三、居住地址核验全套材料
1. 近三个月水电缴费账单、燃气账单、物业缴费单任选其一，必须包含本人姓名、完整居住楼栋房号。
2. 正式房屋纸质租赁合同，包含出租人、承租人双方签字、房屋详细地址、租赁起止日期、物业盖章。
3. 近两个月正规快递收货截图、小区门禁登记记录，地址必须与填报住址完全一致。
4. 异地跨城市申请客户，必须额外提供当地居住证、暂住登记凭证，缺一不可。
5. 合租人群必须补充合租关系说明、共同居住人实名信息备案。

## 四、上班族专项工作证明材料
1. 企业正规纸质在职证明，加盖公司行政公章或人事专用章，含姓名、入职日期、岗位、固定月收入、公司固定座机电话。
2. 连续六个月完整银行代发工资流水，必须显示对公代发备注、企业账户转账记录，无断缴、无空白。
3. 连续六个月社保缴存明细或公积金缴存明细，官网可查，单位缴存正常，无停缴、无补缴异常。
4. 实体工牌实拍照片，含企业LOGO、姓名、部门、工号、办公现场背景。
5. 企业办公场地实景照片，前台、办公区实拍，用于辅助核验真实就业。

## 五、个体经营商户专项经营材料
1. 正规营业执照原件实拍，注册时间满12个月以上，经营范围无高风险禁入行业。
2. 近六个月对公账户经营流水、个人经营进出账流水，交易频次稳定，经营体量正常。
3. 门店门头实拍、店内经营实景、货架设备实拍，真实核验经营场地。
4. 近三个月税务开票记录、纳税凭证任选其一，辅助核验经营真实性。
5. 经营场地租赁合同、房租缴费凭证完整备查。

## 六、收入增信辅助材料（可自愿提交）
1. 个人年度个税完税证明、纳税明细清单。
2. 本人名下房产不动产权证、车辆行驶证、机动车登记证书。
3. 银行定期存款、理财持仓、基金资产、国债资产截图。
4. 过往正规信贷历史结清证明，无逾期良好履约记录。
5. 公积金高额度缴存明细，可直接提升授信审批通过率。

## 七、合规签署类法定材料（必须全部签署）
1. 个人征信查询授权书，本人电子亲笔签名，授权平台查询人行征信、**第三方大数据**。
2. 贷款用途合规承诺书，承诺资金不流入股市、楼市、期货、**非法赌博**、**非正规借贷**、**数字资产**。
3. 借款服务协议、分期还款协议、综合费率告知书，全部阅读确认并签署。
4. 个人隐私信息保护授权协议，合规采集、加密存储、仅用于**授信审核**使用。
5. 贷后回访电话知情同意书，承诺保持电话畅通，配合**合规**回访核验。

## 八、材料驳回直接拒贷规则
1. 材料模糊、反光、遮挡、翻拍、PS修图，直接驳回。
2. 工作单位查无此人、电话空号、地址虚假，直接拒贷。
3. 流水不实、收入虚高、信息前后不一致，拉入**禁入名单**。
4. 拒不配合人脸核验、拒不补充材料，直接终止审批。
5. 短时间反复多次重复上传不合格材料，触发**系统异常监测**。

## 九、优质客户免简化材料政策
1. 平台老用户历史还款良好、无任何逾期，复借可免工作流水、免居住材料。
2. 公积金连续缴存一年以上优质客群，可免人工核验，系统自动审批。
3. 高净值资产白名单客户，**快速审批**，减少全部纸质材料上传流程。

## 十、特殊人群直接免审拒绝
1. 在校全日制学生一律不受理，直接禁止准入。
2. 无业、失业、灵活无固定收入人群直接拒贷。
3. 高危行业、禁入行业人员直接不予审核。
4. 异地无居住证明直接驳回，不予通过。
